TMat2 Math3D

A Matrix which can be either 2 by 2, 3 by 3 or 4 by 4. It can be used for transformations.

The array m is for directly accessing the fields of the matrix. You can also use the properties for accessing columns and rows which consist of an "_" and one number for the row and another for the column. So if you want to let's say access row 0 and column 3 you use _03. The array is stored in row-major order but it only matters if you pass it to another library like OpenGL or Direct 3D or when you use the helper functions for generating a matrix.
